Análise de Dados por Fuzzy

Quando informado, o sistema estima uma prioridade operacional por inferência fuzzy, explicando quais variáveis ativaram a recomendação.

Você pode preencher apenas o que tiver. Campos vazios são ignorados na inferência.

No ISPC, as 15 variáveis são necessárias para calcular o índice. O sistema normaliza automaticamente para 0–10 e aplica o modelo triangular.

No ISPC reduzido, o sistema estima automaticamente: DMP, RMP, Densidade, N de espigas comerciais/ha e Peso de espigas comerciais/ha (calibração 0–10 cm). Para máxima fidelidade, use o modo completo (15 variáveis).
